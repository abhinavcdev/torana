# Security Policy

caddyrs is a network-facing proxy, so security reports are taken seriously.

## Supported versions

Only the latest release receives security fixes.

## Reporting a vulnerability

Please **do not open a public issue** for security problems. Instead, use GitHub's [private vulnerability reporting](https://docs.github.com/en/code-security/security-advisories/guidance-on-reporting-and-writing-information-about-vulnerabilities/privately-reporting-a-security-vulnerability) on this repository, or email the maintainer at chabhinav.95@gmail.com.

Please include:

- A description of the issue and its impact
- Steps to reproduce (a config file and request sequence is ideal)
- The version or commit you tested

You should receive an acknowledgment within a few days. Please allow a reasonable window for a fix to be released before public disclosure.
